This guide covers all available export functions and how to use them in your scripts.

## Available Export Functions

| Function | Description | Returns |
|----------|-------------|---------|
| `GetDiscordAvatar` | Get player's Discord avatar URL | string/nil |
| `GetDiscordRoles` | Get player's Discord role IDs | table |
| `GetDiscordUsername` | Get player's Discord username#discriminator | string/nil |
| `HasDiscordRole` | Check if player has specific role | boolean |

## GetDiscordAvatar

Get a player's Discord profile picture URL.

### Syntax
```lua
exports.Lofi_DiscordAPI:GetDiscordAvatar(playerId)
```

### Example
```lua
local playerId = 1
local avatarUrl = exports.Lofi_DiscordAPI:GetDiscordAvatar(playerId)

if avatarUrl then
    print("Player's avatar: " .. avatarUrl)
else
    print("No avatar available")
end
```

## GetDiscordRoles

Get all Discord role IDs that a player has.

### Syntax
```lua
exports.Lofi_DiscordAPI:GetDiscordRoles(playerId)
```

### Example
```lua
local playerId = 1
local roles = exports.Lofi_DiscordAPI:GetDiscordRoles(playerId)

print("Player has " .. #roles .. " Discord roles")
for i, roleId in ipairs(roles) do
    print("Role: " .. roleId)
end
```

## GetDiscordUsername

Get a player's Discord username with discriminator.

### Syntax
```lua
exports.Lofi_DiscordAPI:GetDiscordUsername(playerId)
```

### Example
```lua
local playerId = 1
local discordName = exports.Lofi_DiscordAPI:GetDiscordUsername(playerId)

if discordName then
    print("Player's Discord: " .. discordName)
else
    print("Discord username not available")
end
```

## HasDiscordRole

Check if a player has a specific Discord role.

### Syntax
```lua
exports.Lofi_DiscordAPI:HasDiscordRole(playerId, roleId)
```

### Example
```lua
local playerId = 1
local vipRoleId = "123456789012345678"

if exports.Lofi_DiscordAPI:HasDiscordRole(playerId, vipRoleId) then
    print("Player is VIP!")
else
    print("Player is not VIP")
end
```

## Practical Examples

### Admin Command System
```lua
local adminRoleId = "123456789012345678"

RegisterCommand('ban', function(source, args, rawCommand)
    if exports.Lofi_DiscordAPI:HasDiscordRole(source, adminRoleId) then
        -- Execute ban command
        print("Admin command executed")
    else
        print("No permission!")
    end
end, false)
```

### VIP Vehicle Access
```lua
local vipRoleId = "987654321098765432"

RegisterNetEvent('vehicle:spawn')
AddEventHandler('vehicle:spawn', function(vehicleModel)
    local src = source
    
    if vehicleModel == "adder" then -- Expensive car
        if not exports.Lofi_DiscordAPI:HasDiscordRole(src, vipRoleId) then
            -- Kick player from vehicle
            print("Player needs VIP for this vehicle")
            return
        end
    end
    
    -- Spawn vehicle
end)
```

### Chat Integration
```lua
RegisterNetEvent('chat:server:SendMessage')
AddEventHandler('chat:server:SendMessage', function(message)
    local src = source
    local playerName = GetPlayerName(src)
    local discordName = exports.Lofi_DiscordAPI:GetDiscordUsername(src)
    
    local displayName = playerName
    if discordName then
        displayName = playerName .. " (" .. discordName .. ")"
    end
    
    -- Send message with Discord info
    TriggerClientEvent('chat:addMessage', -1, {
        args = {displayName, message}
    })
end)
```

## Important Notes

<Warning>
Discord data takes 3-5 seconds to load after a player connects. Add delays when checking Discord data on player join.
</Warning>

### Wait for Data to Load
```lua
AddEventHandler('playerJoining', function()
    local src = source
    
    -- Wait for Discord data
    Citizen.CreateThread(function()
        Citizen.Wait(5000) -- Wait 5 seconds
        
        local hasVIP = exports.Lofi_DiscordAPI:HasDiscordRole(src, vipRoleId)
        -- Now this will work correctly
    end)
end)
```

### Script Load Order
Make sure Lofi_DiscordAPI loads first in your server.cfg:

```cfg
ensure Lofi_DiscordAPI
ensure your_other_script
```

### Always Check for nil
```lua
-- Always check if data exists
local avatar = exports.Lofi_DiscordAPI:GetDiscordAvatar(playerId)
if avatar then
    -- Use avatar
else
    -- Handle no avatar case
end
```

<Info>
All functions return nil/false if Discord data isn't loaded yet or if the player doesn't have Discord linked.
</Info>
