# VIP System

A comprehensive VIP membership system for FiveM that integrates with Discord roles to provide exclusive features, perks, and privileges to your community members.

<Note>
  **Free & Open Source** - Completely free to use and modify for your server.
</Note>

## What It Does

Transform Discord roles into in-game advantages:
- Automatic VIP verification through Discord
- Exclusive features for VIP members
- Tiered pricing system (VIP vs VIP+)
- Built-in VIP shop
- Quick travel teleportation
- Smart cooldown system

## How It Works

<Steps>
  <Step title="Install Dependencies">
    Install ox_lib and JX Discord API
  </Step>
  <Step title="Configure Roles">
    Set your Discord VIP role IDs in the config
  </Step>
  <Step title="Customize Features">
    Adjust prices, cooldowns, and available features
  </Step>
  <Step title="Players Get Access">
    Players with VIP roles automatically get access!
  </Step>
</Steps>

**For Players**: Join Discord → Get VIP role → Press F5 in-game → Enjoy VIP features!

## VIP Features

<Tabs>
  <Tab title="Basic VIP" icon="star">
    Features available to all VIP members:

    - **Health Restoration** - Instant 100% health
    - **Armor Restoration** - Instant 100% armor
    - **Vehicle Repair** - Fully repair current vehicle
    - **Vehicle Wash** - Clean your vehicle
    - **Clothing Menu** - Quick skin/clothing access
    - **VIP Shop** - Discounted item prices
  </Tab>

  <Tab title="VIP+ Exclusive" icon="crown">
    Additional features for VIP+ members:

    - **All VIP Features** - At reduced prices
    - **God Mode** - Temporary invincibility (5 min)
    - **Invisibility** - Become invisible (3 min)
    - **Quick Travel** - Teleport to locations
    - **Better Discounts** - Bigger shop discounts
    - **Exclusive Items** - VIP+ only shop items
  </Tab>

  <Tab title="VIP Shop" icon="cart-shopping">
    Exclusive shop with tiered pricing:

    - Different prices for VIP vs VIP+
    - Configurable item catalog
    - Exclusive items for higher tiers
    - Supports cash, bank, or black money
  </Tab>
</Tabs>
